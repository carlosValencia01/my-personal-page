¿Comó crear una API?

En este tutorial veremos como crear una API utilizando las tecnologias Node.js y MongoDB para ocuparnos de la base de datos.
El framework utilizado para Node.js será Express
Primeramente definamos que es una API: una API es una Interfaz de Programación de Aplicación. Las API permiten que sus productos y servicios se comuniquen con otros proyectos, sin necesidad de saber cómo están implementados.
También existen las API RESTful, este tipo de API se caracteriza por implementar todos los verbos HTTP, los cuales son: (GET, POST, PUT y DELETE). En este tutorial nos centraremos en realizar una API RESTful.
